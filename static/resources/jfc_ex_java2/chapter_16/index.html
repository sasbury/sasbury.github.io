Chapter 16 Example

 
Touch Screen Model
TouchScreenModel.java defines the model for a touch screen object.
 
Touch Screen UI
TouchScreenUI.java defines the UI for a touch screen object.
 
Touch Screen
TouchScreen.java defines the TouchScreen component using the interfaces for the model and UI. The updateUI method has been updated to always use the default UI class, this fixes a bug with in the example that began appearing with recent versions of JFC.
 
Basic Touch Screen UI
BasicTouchScreenUI.java defines the a default TouchScreenUI implementation.
 
Touch Screen Model Imp
TouchScreenModelImp.java defines the a default TouchScreenModel implementation.
 
Touch Screen Test
TouchScreenTest.java defines test program that displays a touch screen.
